# isme-go

> 基于 `gin` `gorm` 开发的兼容 [vue-naive-admin 2.0版](https://github.com/zclzone/vue-naive-admin) 的golang版本服务端

## 在线体验
- [isme](https://admin.isme.top/)

# 本地运行

> 后端

```bash
go run main.go
```

> 前端

```bash
cd vue-naive-admin
npm install
npm run dev
```
